# Aplicación WP7 para consultar paradas de bus, bizi, y otros servicios

 * Patrón MVVM utilizando MVVM Light. 
 * Contenedor de inyección de dependencias para resolver las dependencias entre servicios.
 * Test unitarios de servicios y ViewModels

## Ejecución en el emulador
Para visualizar correctamente la aplicación en el emulador, es recomendable establecer previamente la ubicación actual del emulador. [Información aquí](http://msdn.microsoft.com/en-us/library/hh202933(v=vs.92).aspx)
